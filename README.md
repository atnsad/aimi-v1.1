# 🎯 AIMI
### Precision & Reflex Trainer

## 📖 معرفی
**AIMI** ابزاری است که با هدف بهبود هماهنگی عصب و عضله، و ارتقای دقت (Accuracy) و سرعت عمل (Reflex) توسعه یافته است. در دنیای رقابتی بازی‌ها، میلی‌ثانیه‌ها سرنوشت‌سازند؛ **AIMI** محیطی کنترل‌شده و چالش‌برانگیز برای تسلط بر این مهارت‌هاست.

## 🕹 نحوه شروع
رابط کاربری بازی برای دسترسی سریع طراحی شده است:

* **آموزش:** دسترسی به راهنمای کامل بازی.
* **بخش تمرینی (Practice Levels):** شامل ۵ سطح مختلف.
    * *نکته:* هر چه شماره مرحله بالاتر باشد، فواصل زمانی کوتاه‌تر و چالش دشوارتر خواهد بود.

## 🎮 گیم‌پلی
روند بازی سریع و بی‌رحم است:
1. **آماده‌باش:** با تایمر ۳ ثانیه‌ای شروع می‌شود.
2. **شکار هدف:** دایره‌ها در مکان‌ها و زمان‌های تصادفی ظاهر می‌شوند.
3. **واکنش:** باید قبل از محو شدن کامل دایره، روی آن کلیک کنید.
4. **پایان بازی:** اگر دایره‌ای را از دست بدهید یا زمان بگذرد، بازی به پایان می‌رسد و امتیاز نهایی شما ثبت می‌شود.

## 🚀 ویژگی‌های کلیدی
- **تصادفی‌سازی هوشمند:** الگوریتم مکان‌یابی دایره‌ها هیچ الگوی تکراری ندارد.
- **سختی پیش‌رونده:** ۵ سطح دشواری برای سنجش دقیق پیشرفت شما.
- **بهینه و سبک:** طراحی‌شده برای کمترین میزان تاخیر (Latency) در ثبت کلیک‌ها.
